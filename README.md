# Parser Armor

Конвертер брони из OptiFine CIT в формат RPT (RPT Resource Pack).

## Возможности

- Конвертация файлов `.properties` из OptiFine CIT в RPT формат
- Автоматическое определение текстур брони
- Поддержка всех стандартных материалов (leather, golden, iron, diamond, netherite, chainmail, copper)
- Drag & Drop интерфейс
- Предпросмотр результатов конвертации

## Использование

1. Откройте `index.html` в браузере
2. Перетащите ZIP-архив ресурспака в область загрузки
3. Нажмите "Convert" для конвертации
4. Скачайте результат кнопкой "Download"

## Требования

- Современный браузер с поддержкой JavaScript
- ZIP-архив ресурспака с OptiFine CIT текстурами

## Структура ресурспака

Ожидаемая структура файлов в ZIP:
```
assets/minecraft/optifine/cit/
  ├── armor_name.properties
  └── ...
```
